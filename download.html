Installation

To install ACFS, first install the Addon Engine for Everything, then download ACFS. Move the DLL file to %AppData%\Everything\Addons, restart Everything.

Troubles with Addon Engine?

You must download Portable version of Everything and run it by administrator permissions. You can download it here:
Portable Everything x86
Portable Everything x64

Note

If you want to edit ACFS settings, open acfs.ini in notepad.
